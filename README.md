# ขั้นตอนการติดตั้ง<br />
1.ติดตั้ง node Js และ npm<br />
2.โหลด Repositories นี้<br />
3.เข้าcmd และใช้ CD เข้าไปที่ createder-test\frontend<br />
4.ใช้คำสั่ง npm install <br />
5.ใช้คำสั่ง npm start<br />
